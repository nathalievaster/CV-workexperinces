Om denna webbplats

Hem | Lägg till | Om

Denna webbplats är en frontend-applikation som konsumerar ett REST API byggt med Express och PostgreSQL. Syftet är att visa, lägga till och hantera arbetserfarenheter.

Databasserver: PostgreSQL

Slutsatser

  • Det är viktigt att strukturera sin kod väl, särskilt när frontend och backend separeras. Det har underlättat enormt att dela upp JavaScript-koden i fler filer.
  • Att använda fetch för att interagera med API:er är inget nytt, men att använda ett eget API var något nytt! Det visade sig vara betydligt enklare än jag kunde tänka mig.
  • I denna applikation har jag sett till att få med mer tydliga felmeddelanden gentemot användaren, istället för att använda mig av ett enda felmeddelande som "Fyll i alla fält".
